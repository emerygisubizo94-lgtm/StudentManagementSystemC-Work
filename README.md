# Student Management System

## Overview
This project is a Student Management System developed in C++.

The system demonstrates:
- Functions
- Structures
- Header Files
- Source Files
- Menu-driven Programming

## Features
- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student

## Technologies Used
- C++
- Object-Oriented Concepts
- Modular Programming

g++ main.cpp student.cpp -o system

./system
